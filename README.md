# まいにちレシピアシスタント

## アプリの目的

**まいにちレシピアシスタント**は、Instagram、TikTok、YouTube、Xなどで見つけたSNSレシピを保存し、あとから料理名・材料・作り方・タグ・メモで素早く探すためのスマホ向けレシピ管理アプリです。

HTML / CSS / JavaScript だけで動く `index.html` 1ファイル構成のため、GitHub Pagesでそのまま公開できます。外部APIは使わず、保存データはブラウザの `localStorage` に保存します。

## GitHub Pages 公開URL

公開後に以下へURLを記入してください。

- 公開URL: `https://<ユーザー名>.github.io/recipe-assistant/`

## 主な機能

- SNSで見つけたレシピの保存
- 保存済みレシピの一覧表示
- 料理名・材料・作り方・タグ・メモでの検索
- 複数キーワード検索
- 保存済みレシピの詳細表示・編集・削除
- お気に入り登録、作った記録、最後に作った日の確認
- JSONバックアップのエクスポート / インポート
- 家族情報メモ
- スマホ向け3タブUI（検索 / 保存 / 家族）
- 手動テストチェックリスト

## 使い方

### 1. 保存済みレシピを検索する

1. 「検索」タブを開きます（初期表示は検索タブです）。
2. 料理名、材料名、作り方、タグ、メモに含まれるキーワードを入力します。
3. スペース区切りで複数キーワード検索できます。
4. 検索欄が空欄の場合は保存済みレシピを全件表示します。
5. カードの「詳細を見る」「編集する」「削除」から保存済みレシピを操作できます。

例:

- `チョコ`
- `小麦粉`
- `キャベツ`
- `チョコ 小麦粉`

### 2. レシピを保存する

1. 「保存」タブを開きます。
2. 「レシピ情報を入力する」を開きます。
3. 料理名、SNS URL、画像メモ、材料、作り方、タグ、メモを入力します。
4. 「保存する」を押します。

※ SNS URLは空欄でも保存できます。料理名は必須です。

### 3. 保存済みレシピを編集する

1. 「検索」タブの保存済みレシピカードで「編集する」を押します。
2. 「保存」タブの編集フォームが開きます。
3. 料理名、材料、作り方、URL、タグ、メモを修正します。
4. 「更新する」を押すと、同じレシピIDのまま `localStorage` に上書き保存され、詳細表示の更新日時も更新されます。

### 4. バックアップする

- 「保存」タブの「バックアップをエクスポート」を押すと、保存済みレシピのJSONを出力できます。
- JSONを貼り付けて「JSONをインポート」を押すと、保存済みレシピを取り込めます。
- JSON形式が正しくない場合は、エラーが表示されます。

## 保存データについて

このアプリはサーバーを使わず、ブラウザの `localStorage` にデータを保存します。

主な保存データ:

- `savedRecipes`: 保存済みSNSレシピ
- `familyInfo`: 家族情報

注意点:

- 同じ端末・同じブラウザ内で保存されます。
- ブラウザのデータ削除や別端末利用ではデータが引き継がれません。
- 必要に応じてJSONエクスポートでバックアップしてください。
- 外部APIやSNSからの自動取得は行いません。
